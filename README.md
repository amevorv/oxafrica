# Ox-Africa

This project is a modern redesign of the Ox-Africa website, a platform for agricultural machinery and solutions. The new design features a clean, responsive layout built with React and Bootstrap.

## Technologies Used

* **React:** A JavaScript library for building user interfaces.
* **Vite:** A fast build tool for modern web projects.
* **React Bootstrap:** A library of reusable front-end components.
* **React Router:** A library for handling routing in React applications.
* **React Icons:** A library of popular icon packs.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

## Usage

Once the development server is running, you can view the website at `http://localhost:5174/`.
